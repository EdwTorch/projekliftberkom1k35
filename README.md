# Projek 1 Mata Kuliah Berpikir Komputasional (Mekanisme Lift) 
Program ini dibuat untuk menerapkan beberapa basic pemrograman ke dalam penerapan dunia nyata berupa mekanisme lift 
## File Source Code dari Projek ini adalah File bernama mainprogramfull.py

## Fitur yang tersedia 
1. Memasukkan lantai awal untuk dimasuki pengguna
2. Validasi Input Lantai
3. Validasi Berat Pengguna
4. Input Berat dan Jumlah Pengguna
5. Mekanisme naik dan turun lift yang terurut

## Fitur yang masih belum terimplementasi 
1. Lift dapat mendapatkan masukan orang dan input orang di setiap lantai
2. Setiap orang dan berat badannya terhubung dengan tujuan lantai yang mereka masukkan sehingga saat mereka turun, beban yang dikeluarkan sesuai dengan orang yang keluar
3. Tombol darurat jika terjadi kebakaran atau gempa bumi atau mati listrik yang membuat semua tujuan awal dilewati dan lantai langsung menuju lantai basement untuk membuka lantai
4. Program dapat berulang terus menerus hingga lift kosong
5. Prioritas input lantai dari tombol luar agar lift bisa berhenti
6. Membuat kode sumber yang lebih rapi dan terstruktur menggunakan function dan prosedur (subprogram).
